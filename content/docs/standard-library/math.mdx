# Math
## Methods
This module provides mathematical functions and constants.

### `Masamu.yochepa`

Returns the smallest of the given two numbers.

```duwa
Masamu.yochepa(1, 2); // 1
```

### `Masamu.sqrt`

Returns the square root of a number.

```duwa
Masamu.sqrt(4); // 2
```
