## Methods

`Khonso.lemba()`

Writes out to the console

```duwa
Khonso.lemba("Awa ndi mawu");
```

`Khonso.fufuta()`

Clears the console

```duwa
Khonso.lemba("Awa ndi mawu");
Khonso.fufuta();
```

`Khonso.landira()`

Reads from the console

```duwa
mawu zowerenga = Khonso.landira();
Khonso.lemba(zowerenga);
```
