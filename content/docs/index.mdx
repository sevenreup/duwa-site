# Welcome to Duwa

## Installation

Download the prebuilt binaries for your platform from the [release](https://github.com/sevenreup/duwa/releases)

Create a new source file, `main.duwa`.

Paste the following cool code

```duwa
lemba("Moni Dziko");
```

Run you new application

```bash
duwa -f ./main.ny
```
